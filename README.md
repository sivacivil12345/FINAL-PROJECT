# 🚍 Transportation Revenue and Expenditure analysis

## 📌 Project Overview
This project presents an end-to-end exploratory data analysis (EDA) and business intelligence dashboard for Transportation funding analysis Dataset. The primary objective of this dataset is to analyze and compare transportation-related public finance statistics across U.S. states over multiple years and derive actionable insights to support data-driven transportation decision-making.

The project combines Python (Google Colab) for data preprocessing and exploratory analysis with Power BI for developing an interactive executive dashboard.

## 🎯 Project Objectives
◆ Clean and preprocess the raw transportation funding dataset.

◆ Perform Exploratory Data Analysis (EDA) to identify patterns, trends, and outliers.

◆ Analyze transportation statics by :

    ◇ State
		
    ◇ Mode of transportation
		
    ◇ Year
		
◆ Study statistical properties such as skewness, kurtosis, and correlations.
